Kaskadni stilovi

Postoji 3 vrste primene kaskadnih stilova na html dokumentima.

  1. Direktno primene stilova kao atributa nekog taga koji zelimo da menjamo.
  2. Unutar head taga imamo sryle tag unutar kog vrsimo primenu CSS_a.
  3. Eksterni CSS. Njega zapravo karakterise novi fajl(npr. style.css) koji ce biti povezan sa nasim html fajlom.
